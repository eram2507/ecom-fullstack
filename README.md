# E-commerce Full Stack Application

## Overview
This project is a full-stack e-commerce application built using Angular and .NET Web API. It includes basic product management and a cart system with quantity tracking and total price calculation.

## Tech Stack
- Frontend: Angular (TypeScript)
- Backend: .NET Web API (C#)
- Architecture: REST APIs

## Features
- View products
- Add new products
- Add items to cart
- Cart with quantity handling
- Total price calculation

## In Progress
- Moving data from in-memory storage to SQL Server
- Backend-based persistent cart

## How to Run

### Backend
cd EcomAPI  
dotnet run  

### Frontend
cd ecom-ui  
ng serve  

Then open http://localhost:4200
